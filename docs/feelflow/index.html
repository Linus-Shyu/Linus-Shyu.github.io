系统要求

  • iOS 18.0 或更高版本
  • iPhone、iPad、Mac 兼容
  • 需要网络连接（用于AI分析功能）
  • API 密钥已内置，无需配置

快速开始

  1. 从 App Store 下载 FeelFlow
  2. 打开应用，开始创建第一篇笔记
  3. 点击"情感洞察"获取AI分析
  4. 使用"心灵回响"获得AI回应
  5. 在主屏幕添加小组件，每日获取格言
